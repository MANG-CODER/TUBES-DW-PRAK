# Mangnime Streaming Anime No Iklan
Mangnime merupakan platform layanan hiburan berbasis streaming, salah satunya adalah konten anime. sebagian besar platform streaming anime yang tersedia saat ini masih menyajikan banyak iklan, keterbatasan akses, serta antarmuka yang kurang ramah pengguna, sehingga mengurangi kenyamanan dalam menonton.
Berdasarkan permasalahan tersebut, kelompok kami mengembangkan sebuah platform streaming anime tanpa iklan yang berfokus pada kenyamanan pengguna dan kemudahan akses informasi. Platform ini dirancang agar pengguna dapat menonton anime secara lebih fokus tanpa gangguan iklan, serta memperoleh informasi detail seperti genre, jadwal rilis, status anime (ongoing atau complete), dan detail episode dengan tampilan yang sederhana dan responsif.

Dalam pengembangannya, website ini dibangun menggunakan teknologi HTML, CSS (Tailwind CSS), dan JavaScript, dengan struktur folder yang terorganisir antara file publik, asset gambar, serta script JavaScript untuk mengatur logika dan interaksi halaman. Setiap halaman memiliki fungsi tersendiri, seperti halaman utama (index), genre, detail anime, jadwal, batch download, serta halaman kebijakan seperti privacy, DMCA, dan terms. Penggunaan JavaScript memungkinkan pengolahan data secara dinamis sehingga informasi dapat ditampilkan secara interaktif tanpa perlu memuat ulang halaman secara berlebihan.

Selain itu, platform ini juga dirancang dengan konsep responsif, sehingga dapat diakses dengan baik melalui berbagai perangkat seperti laptop, tablet, maupun smartphone. Dengan adanya proyek ini, diharapkan platform streaming anime yang dibuat dapat menjadi alternatif layanan hiburan digital yang lebih nyaman, ringan, dan mudah digunakan, sekaligus menjadi sarana penerapan dan pengembangan kemampuan mahasiswa dalam bidang desain web dan pemrograman front-end.
